Qui modifie quoi ?

  • Vous, sans code — contenu (textes, prix, FAQ, images) via le CMS ; pilotage bêta via le cockpit ; clés/secrets via Cloudflare.
  • Claude (sur demande) — code : design, agents IA, espace membre, nouveaux outils. Claude pousse d'abord sur le brouillon (staging), vous validez, puis on publie.

👉 Pour une modif de code, briefez Claude simplement : « Sur staging : change X ». Vous prévisualisez, puis vous cliquez Publier.

Publier le brouillon en 2 clics

Quand le site brouillon vous convient :

  • Cliquez « Publier le brouillon » ci-dessus (ouvre GitHub : comparaison staging → main).
  • Bouton vert « Create pull request » puis « Merge » → le site en ligne se met à jour en ~1 min.
🚀 Publier maintenant

Aucune modif en attente ? GitHub affichera « There isn't anything to compare ». Normal.
