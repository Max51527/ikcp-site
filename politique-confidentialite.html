II. Données collectées

2.1 Données collectées lors de la navigation

  • Adresse IP (anonymisée — Cloudflare) · User-agent

2.2 Données collectées lors de l'inscription (espace bêta)

  • Nom, prénom, email professionnel
  • Estimation du patrimoine (déclaratif, fourchette)

2.3 Données traitées par Marcel (agents IA)

  • Questions et contexte transmis aux agents : traitement en mémoire, non stockés de manière permanente
  • Audit log horodaté : question anonymisée + empreinte MIF II — stocké D1 Paris (90 jours)

VII. Cookies

IKCP n'utilise pas de cookies tiers de tracking ni de publicité comportementale. Les seuls cookies présents sont :

  • Session authentification (magic link) : durée de session, httpOnly, Secure, SameSite=Strict
  • Préférences interface (thème, langue) : localStorage, données restent sur votre appareil

Aucun bandeau de consentement cookies car aucun cookie tiers n'est déposé.

VIII. Sécurité

Les données sont protégées par :

  • Chiffrement TLS 1.3 en transit (Cloudflare)
  • Chiffrement au repos (D1 SQLite — Cloudflare)
  • Authentification par magic link uniquement (pas de mot de passe stocké)
  • Audit log MIF II horodaté pour tout accès aux données patrimoniales
  • Secrets API jamais dans le code (Cloudflare Secrets API)
