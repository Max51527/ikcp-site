# IKCP — Analyse concurrentielle Family Offices

> **Objet** : carte du marché des family offices et acteurs adjacents
> (FR + international + SaaS + fintechs), positionnement IKCP, gaps
> exploités, menaces à surveiller.
>
> **Statut** : v1 · 09/05/2026 · sources publiques uniquement
> **Méthode** : recherche structurée (WebFetch + WebSearch) sur 26
> acteurs identifiés. Aucun chiffre privé, aucune extrapolation.
> **Documents associés** : `FO-DEFINITION-MANIFESTO.md`,
> `FAMILY-OFFICE-AGENTS-AUDIT.md`

---

## 0. Méthode et limitations

- **Sources** : sites publics + presse spécialisée (Décideurs, CFNEWS,
  Profession CGP, Gestion de Fortune, WealthBriefing) + grilles tarifaires
  publiées + études de marché publiques.
- **Sites bloqués lors du scrap** : cyrusconseil.com, equance.com,
  oddo-bhf.com (HTTP 403). Données reconstituées via sources tierces.
- **Tarifs % AUM** quand cités sont **issus de la presse spécialisée**
  (familyoffice-france.fr, Andsimple, Masttro guide), pas des prix
  officiels publiés des acteurs.
- **Maturité digitale** est une note subjective 0-5 basée sur : qualité
  site / portail client / app mobile / IA conversationnelle / agrégation
  temps réel. **0** = vitrine simple, **5** = plateforme complète + IA.

## 2. Synthèse exécutive — 3 patterns dominants

### Pattern 1 — Tarification opaque chez les FO traditionnels

**Aucun** des 9 FO français traditionnels analysés (Cyrus Herez, Wormser,
Equance, ODDO, Compagnie 1818/Natixis WM, Witam, Lazard, Pictet, Edmond
de Rothschild) ne publie ses tarifs. La logique reste **% AUM** (0,4 -
1,5 % selon la presse spécialisée) avec ticket implicite ≥ 10-25 M€.

À l'inverse, **fintechs FR + Masttro + Stableton** publient leurs grilles
en clair. **Finary** affiche 149,99 €/an pour son tier Plus.

→ **Gap** : il n'existe pas, en France, de FO digital avec tarif
forfaitaire transparent < 10 k€/an.

### Pattern 2 — Forte maturité digitale chez les SaaS US et fintechs FR · faible chez les FO traditionnels FR

| Catégorie | Maturité moyenne |
|---|:---:|
| FO français traditionnels | 2,2 / 5 |
| FO internationaux | 3,3 / 5 |
| **SaaS FO US** | **4,4 / 5** |
| **Fintechs FR** | **4,3 / 5** |

→ Northern Trust (Wealth Passport), Eton (AtlasFive WealthAI), Masttro
(Masttro Intelligence) intègrent **IA conversationnelle native**. Les
FO français restent à 1-3/5 — sites vitrine, peu d'apps client,
**aucun chatbot grand public identifié**.

→ **Gap** : aucun FO français n'a déployé d'agent conversationnel
client final. L'IA est cantonnée aux SaaS de back-office côté pro.

### Pattern 3 — NextGen est un quasi-désert digital

Sur les 26 acteurs analysés :
- **3 FO/réseaux** ont un programme NextGen : Bessemer Trust (éducation
  financière), Lombard Odier (LO Generations), FOX (Rising Generation
  Leadership)
- **Tous trois sont événementiels/présentiels** — sommets, workshops,
  peer learning. **Pas de plateforme digitale produit**.
- **AFFO × EDHEC** propose un présentiel certifiant à 6 500-7 500 €
  depuis janvier 2025
- **Aucun** SaaS, aucune fintech, aucune banque privée n'a une
  plateforme digitale dédiée aux **enfants** de dirigeants d'entreprise
  familiale

→ **Gap majeur** : c'est précisément ce que la **Formation NextGen
IKCP** (6 modules digitaux personnalisés sur le patrimoine réel) vient
combler — premier acteur sur ce créneau.

## 4. Les 4 gaps qu'IKCP exploite

### Gap 1 — Forfait transparent < 10 k€/an pour HNW intermédiaire

**Marché inadressé** : dirigeants TPE/PME/ETI avec patrimoine 1-10 M€.
**Volume estimé** : ~50 000 dirigeants en France selon les statistiques
INSEE 2023 sur les entreprises familiales 1-10 M€ de CA.

**IKCP TPE 2 400 €/an** = ouvre cette cible massive.

### Gap 2 — IA conversationnelle client final

Aucun FO français ne propose Marcel-équivalent. Seuls les **SaaS B2B US**
(Eton, Masttro) intègrent de l'IA, mais côté outils internes pour les
family officers — pas pour les familles elles-mêmes.

**IKCP** = premier produit français avec IA orchestrateur côté client,
sourcé Légifrance/BOFIP.

### Gap 3 — NextGen 100 % digital personnalisé

**Aucun acteur** ne propose un parcours digital *bâti sur le patrimoine
réel de la famille*. AFFO × EDHEC fait du présentiel certifiant
générique, LO Generations fait de l'événementiel.

**IKCP NextGen** = parcours sur les chiffres réels (vue 360° → fiscalité
familiale → pacte d'actionnaires → 4 schémas transmission → charte → plan
10 ans + certification).

### Gap 4 — Code à vous, exportable

Les FO traditionnels et SaaS verrouillent les données dans un format
propriétaire. **Quitter = repartir nu**.

**IKCP** = endpoint `/api/export/me` → JSON complet horodaté SHA-256 en
1 clic. Argument anti-friction à l'adhésion.

## 5. Les 3 menaces à surveiller

### Menace #1 — AFFO × EDHEC NextGen (lancée janvier 2025)

| Critère | Détail |
|---|---|
| Tarif | **6 500 - 7 500 €** (vs IKCP Premium 6 800 €/an) |
| Marque | AFFO + EDHEC = légitimité académique très forte |
| Format | 7 modules · 6-10 mois · hybride · certifiant |
| Risque | **Si AFFO/EDHEC digitalise** son programme = concurrent direct frontal |

**Mitigation IKCP** :
- Vitesse d'exécution (beta lance S+3, leur digitalisation prendra 18-24 mois)
- Hyper-personnalisation sur les chiffres réels (AFFO/EDHEC restera
  générique académique)
- Inclusion Marcel + dashboard + écosystème complet (pas juste formation)
- Possibilité de **partenariat** AFFO/EDHEC : IKCP délivre la
  pédagogie, AFFO certifie

### Menace #2 — Finary One (gestion privée + série B 25 M€ PayPal Ventures sept. 2025)

| Critère | Détail |
|---|---|
| Trajectoire | Agrégateur 850 k users → gestion privée Finary One (50 M€ AUM) |
| Tarif | 149,99 €/an Plus + Finary One commissionné |
| Capital | Série B PayPal Ventures sept. 2025 |
| Risque | **Si Finary lance un volet famille / NextGen**, capture de l'audience massive |

**Mitigation IKCP** :
- Finary cible le particulier, IKCP cible les familles dirigeantes — segments distincts
- Finary n'a pas de Maxime — pas de validation MIF II humaine
- IKCP est CGP CIF + COA, légitimité réglementaire vs Finary fintech
- Possibilité de **complémentarité** : Finary pour l'agrégation
  particulière, IKCP pour la couche famille / NextGen / Maxime

### Menace #3 — White-labelisation Masttro / Eton Solutions

| Critère | Détail |
|---|---|
| Risque | Si Masttro/Eton white-labelisent leur IA conversationnelle pour FO français |
| Probabilité | Moyenne — leur business actuel est B2B back-office, pivot front client coûteux |
| Délai estimé | 12-24 mois si volonté stratégique |

**Mitigation IKCP** :
- Marcel est entraîné/contextualisé sur le **droit français** (CGI, BOFIP,
  conventions FR-XX) — pas un wrapper générique
- Marque IKCP + Maxime + 10 ans d'expertise terrain
- Stack 100 % Cloudflare-first souverain EU vs SaaS US

## 6. Recommandations stratégiques

### Court terme (S0 → S+3, avant lancement beta)

1. **Position défensive AFFO/EDHEC** : prendre contact avec AFFO pour
   explorer un partenariat (leur certification académique + IKCP
   pédagogie digitale personnalisée). Win-win possible.
2. **Communication différenciante** sur les 4 gaps : forfait transparent,
   IA client, NextGen digital personnalisé, code à vous.
3. **Tracking concurrentiel** : Google Alert sur "AFFO EDHEC NextGen",
   "Finary One famille", "Masttro family", "Eton WealthAI front client".

### Moyen terme (Phase 2-3 beta)

4. **Différenciation par profondeur** : continuer à empiler les sub-agents
   MCP (juridique RAG, sub-agents univers art/vins/voitures) — chaque
   ajout creuse l'écart vs concurrents.
5. **Partenariats régulateurs** : adhésion AFFO comme membre actif
   (~3-5 k€/an) pour positionner IKCP comme acteur du marché FO français.
6. **Open-sourcing partiel** : open-sourcer un MCP server (ex:
   `documents-mcp-server` français-spécialisé) pour positionner IKCP
   comme acteur technique de référence.

### Long terme (post-beta, S+24+)

7. **Partenariat Finary One** envisageable : IKCP devient le tier
   "famille/NextGen" de Finary (réseau 850 k users → IKCP capture les
   1-5 % qui sont dirigeants familiaux).
8. **Levée de fonds** ciblée : si Finary One a levé 25 M€ pour une
   trajectoire fintech B2C, IKCP peut viser ~3-5 M€ pour scaler le modèle
   créateur de FO digital sur 200-500 familles + 20-50 cabinets B2B.
