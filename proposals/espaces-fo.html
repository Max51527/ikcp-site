    Espace client connecté magic-link · pas de mot de passe. Vous saisissez votre email, vous recevez un lien, vous êtes connecté pour 30 jours. Sécurité bancaire, friction zéro.

    Photo → coffre-fort instantané · vous recevez un courrier RAR DGFiP : photo depuis le téléphone, upload R2, classement automatique par Documents-agent, rappel programmé par Suivi-agent.

    Mode hors-ligne · 4 réponses ultra-fréquentes (donation, IFI, AV, barème IR) accessibles sans réseau. Service worker cache les routes essentielles.

    Export RGPD 1 clic · vos données vous appartiennent. Bouton « Exporter mes données » dans le dashboard → JSON complet horodaté SHA-256, téléchargé instantanément.

    Sub-comptes famille · les enfants ont leur propre vue, leurs propres modules NextGen, leurs propres conversations. Vous voyez leurs progrès en temps réel.
