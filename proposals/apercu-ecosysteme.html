~30 min
ii.
D1 schema · wrangler d1 execute ikcp-client-db --file=schema.sql --remote (10 tables Phase 1+2)
~5 min
iii.
R2 buckets · wrangler r2 bucket create ikcp-docs-private + ikcp-templates + ikcp-archives
~10 min
iv.
Secrets · ANTHROPICAPIKEY · RESEND_API_KEY · MCP_SHARED_SECRET (généré 1× et partagé sur les 3 workers concernés) · INSEE_KEY (optionnel) · NOTION_TOKEN (optionnel)
~15 min
v.
50 codes beta · node scripts/generate-beta-codes.mjs --count 50 --source "cohorte-1" puis insérer le SQL généré dans D1
~15 min
vi.
DPA Anthropic · email à legal@anthropic.com demandant le DPA Enterprise (data processing addendum + Standard Contractual Clauses)
1 sem négo
vii.
Sélection cohorte 1 · 5 prospects qualifiés (TPE classique · ETI · holding immo · frontalier · solo+enfant 25 ans) · visios pré-qualification 30 min
2-3 j Maxime
viii.
Smoke tests · health endpoints + flow upload doc + flow chat Marcel + flow beta-redeem · valider sur compte test avant J0
~2 h
