Architecture

Trois couches. 100 % France.

Tout transite par Cloudflare Workers EU sovereign (edge) + D1 (SQLite UE) + Stripe + Resend (Anthropic via DPA). Aucune donnée nominative ne traverse les calculateurs.

Infrastructure technique

○Compte Resend + DNS ikcp.eu vérifié
○D1 ikcp-client-db créée + schema appliqué
○Worker ikcp-client déployé · magic link OK
○Worker ikcp-pappers déployé · KV cache 1h
○DNS client.ikcp.eu → worker
○Pages /login · /salon codées et live
○Tests E2E : signup → magic → dashboard → lookup
Livraison J+10 Cinq utilisateurs internes peuvent s'inscrire, parler à Marcel, faire un lookup Pappers. Pas encore de paiement.
